###connectSocket 　 header 参数类型不同; protocols没找到同名参数; tcpNoDelay没找到同名参数 　 可选入参 不一样

  • 百度小程序原文档地址 创建一个 WebSocket 连接

###onSocketOpen wx 有，ali 没有 wx 和 swan缺少 paramsAttribute 　 支付宝未提到返回值 header
